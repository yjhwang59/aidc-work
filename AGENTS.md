# AIDC.work Agent Instructions

## 1. Project Goal

AIDC.work is the professional website of Jack Y. J. Hwang, an Enterprise AI Development Consultant.

The website presents Jack's work in:

- AI tool training
- Vibe Coding
- AI system development
- AI Agent workflow implementation
- Enterprise AI adoption roadmap
- Consultant templates, cases, and blog articles

AIDC.work is not only a personal website. It is a long-term consultant asset library for articles, cases, templates, workflows, and AI-assisted development examples.

## 2. Brand Positioning

Primary positioning:

Jack Y. J. Hwang helps enterprises turn AI tools, vibe coding, and AI agents into practical systems and workflows.

Chinese positioning:

協助企業從 AI 工具使用，走向 AI 系統開發與 AI Agent 工作流落地。

Brand tone:

- Professional
- Practical
- Trustworthy
- Clear
- Consultant-like
- Not overly flashy
- Not buzzword-heavy

Primary language:

- Traditional Chinese

Secondary language:

- English may be used for technical terms, professional titles, and bilingual headings.

## 3. Tech Stack

Use the following stack for MVP:

- Next.js App Router
- TypeScript
- Tailwind CSS
- MDX / Markdown content
- Static-first architecture
- No database in MVP
- No authentication in MVP
- No CMS in MVP

Do not introduce unnecessary dependencies.
